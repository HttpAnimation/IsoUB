IsoUB is software only ment to be ran on Linux and not MacOS or Windows so do hot run the scripts on Windows or MacOS it also made to be used while you already have grub boot manager installed and working.

Installing on Debain 12

Download the install script with either curl or wget.

            
                wget https://exp.com
                curl https://exp.com

Now allow the script to run in the same terminal run.

            
                chmod +x Install-Debian.sh

Once you have everything downloaded and allowed to run you are now free to run the script to do so run the command.

            
                bash Install-Debian.sh

The script will auto delete it's self and IsoUB will be installed.

To run the IsoUB this also goes for every other distro run in the terminal.
